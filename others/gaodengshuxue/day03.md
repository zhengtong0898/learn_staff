
# 集合  
相同类型的组就是一个集合.  

&nbsp;  
### 集合符号  
**空集:** $ \emptyset $  
**无穷集** $ \infty $  

&nbsp;  
### 集合的关系  
**全集:** $ A \cup B $  
**包含:** $ \displaylines{ 
               \\\\
              \ \ \ \ \ \ \  A \subset \cup   \ \ \ \ \ \ \ \ \ \  // 表示A是U的子集  
               \\\\
               \cup \subset A   \ \ \ \ \ \ \ \ \ \  // 表示U包含A
            }
          $
