
# 二元一次方程组  
**二元:** 两个未知数, 通常是 x 和 y.  
**一次:** 未知数不带平方.  


&nbsp;  
### 判断是否有解
&nbsp;  
> **有无穷多的解:** $ {a_1 \over a_2} = {b_1 \over b_2} = {c_1 \over c_2} $

&nbsp;  
> **有唯一解:** $ {a_1 \over a_2} \ne {b_1 \over b_2}  $  

&nbsp;  
> **无解:** $ {a_1 \over a_2} = {b_1 \over b_2} \ne {c_1 \over c_2} $  



&nbsp;  
### 解方程  

> 代入消元法  

- **例-1**  
  $
  \begin{align}
      \begin{cases}
           x   +   y  = 3 \\\\\\\\
          {x \over y} = 5  
      \end{cases}
  \end{align}
  $  
  &nbsp;  
  **判断是否有解**  
  暂无  
  &nbsp;  
  **解方程**
  转换过程: $ x = 3 - y $     
  代入消元: $ { 3 - y \over y } = 5 $  
  过程: $ 3 - y = 5y $  
  过程: $ 3 = 5y + y $  
  过程: $ 3 = 6y $  
  过程: $ {3 \over 6} = {6y \over 6} $  
  过程: $ {1 \over 2} = y $  
  结果(y): $ y = {1 \over 2} $  
  过程: $ x + {1 \over 2} = 3 $  
  过程: $ x = 3 - {1 \over 2} $  
  过程: $ x = {6 \over 2} - {1 \over 2} $  
  结果(x): $ x = {5 \over 2} $  
  结论: $ \begin{cases}
              x = {5 \over 2} \\\\\\\\ 
              y = {1 \over 2}  
          \end{cases}
        $
