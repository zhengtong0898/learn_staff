### 标签(指标)管理  
标签管理的初级阶段秉持的原则是: 世上本无路, 走着走着路就出来了.  
标签也是一样的, 给问题打标签最开始也是没有标签的, 遇到一个问题就总结好打一个标签,     
时间长了不同的维度的标签就出来了, 当标签多了之后就可以对标签进行分类管理了.
